# 🏥 MedFlow

## 📌 Sobre o MedFlow
O **MedFlow** é um sistema de gestão de agendas para consultórios médicos e estéticos, desenvolvido para facilitar o agendamento de consultas. A plataforma permite que pacientes marquem, reagendem e cancelem consultas de forma simples, enquanto profissionais gerenciam sua disponibilidade e organizam seus atendimentos de maneira eficiente.
