# 🏥 MedFlow

## 📌 Sobre o MedFlow
O **MedFlow** é um sistema de gestão de agendas para consultórios médicos e estéticos, desenvolvido para facilitar o agendamento de consultas. A plataforma permite que pacientes marquem, reagendem e cancelem consultas de forma simples, enquanto profissionais gerenciam sua disponibilidade e organizam seus atendimentos de maneira eficiente.

# 🟡 SPRINT 1 - Acesso e Agendamento Inicial

---

### 🧩 US-01 - Login
**Como** usuário (paciente ou profissional)
**Quero** realizar login com e-mail e senha  
**Para** acessar meu painel de forma segura

**Prioridade:** ALTA  
**Story Points:** 5  

---
### 🧩 US-02 - Fluxo de Agendamento
**Como** paciente  
**Quero** visualizar a lista de médicos e acessar suas agendas  
**Para** escolher o profissional e marcar uma consulta. 

**Prioridade:** ALTA  
**Story Points:** 7    

---

### 🧩 US-03 - Gestão de Disponibilidade
**Como** profissional
**Quero** configurar meus dias e horários de atendimento  
**Para** que os pacientes saibam quando estou disponível para consultas. 

**Prioridade:** ALTA  
**Story Points:** 8
